### 👨‍💻 About Me

I love bridging the gap between clean software and physical hardware. Whether it's crafting a sleek UI in Android or debugging a custom circuit board, I enjoy building things that actually work (eventually). 

*   📱 **Currently building:** Android applications using **Kotlin & Jetpack Compose** (including background tasks and Material 3 design).
*   🔌 **Currently tinkering with:** **ESP32-S3** microcontrollers, I2C expanders, and building custom MIDI peripherals (like a 32-note virtual pipe organ pedalboard).
*   🏠 **Past projects:** Smart home assistants integrating **Python, Flask, and SQLite**.
*   🎹 **When I'm not coding:** You can find me playing the piano, following Jesus, or cheering for HK Nitra 🏒.
